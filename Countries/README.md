# The Project Brief: IPTV Logo Collection

### What is this?
This project is a personal mission to build a comprehensive, organized library of IPTV channel logos categorized by country. High-quality, consistent logos make any media player look better, and I noticed how hard it can be to find them all in one place, so I'm building that place for myself and anyone interested.

### Project Structure
To keep things clean, the repository is organized by geographic region:

* **Folders by Country:** Each country has its own dedicated folder (e.g., /USA, /UK, /Canada).
* **File Naming:** I aim for clear naming conventions so logos are easy to map to your .m3u playlists.
